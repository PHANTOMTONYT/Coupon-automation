PRD: Amazon coupon verification POC using Playwright

PRD — POC

Amazon coupon code verifier

v0.1 — POC Playwright + Python 1 night build

Objective

Build a proof-of-concept that takes an Amazon coupon/promo code as input, navigates to a hardcoded product, applies the code at cart level, and returns whether the code is valid or invalid — along with the discount value if valid.

Scope

In scope

  • Single hardcoded ASIN (amazon.in)
  • Manual pre-login via real browser
  • Cookie injection into Playwright
  • Add to cart flow
  • Apply promo code at cart
  • Parse success / error message
  • Return structured result (valid, discount amount)

Out of scope

  • Automated login / CAPTCHA handling
  • Multiple products or ASINs
  • Proxy / stealth infra
  • Bulk code verification
  • Actual order placement
  • Multi-country support
  • Retry / error recovery logic

Happy path flow

1

Load cookies

Export Amazon session cookies manually → inject into Playwright context

2

Navigate to product

Go to hardcoded ASIN URL on amazon.in

3

Add to cart

Click 'Add to Cart' → confirm cart page loads

4

Open cart

Navigate to cart → verify item is present

5

Apply coupon

Find promo code input → fill code → click apply

6

Parse result

Read success message (discount applied) or error message (invalid code)

7

Return output

Print structured result: { valid: bool, code: str, discount: str | null }

Expected output

# valid code
{ "code": "SAVE20", "valid": true, "discount": "₹200 off", "message": "Promo code applied" }

# invalid code
{ "code": "FAKE99", "valid": false, "discount": null, "message": "Invalid promo code" }
